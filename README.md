# 한국복지패널 데이터 분석 시각화 대시보드

이 프로젝트는 **한국복지패널 데이터**를 활용하여 성별, 연령, 직업 등에 따른 경제적/사회적 지표를 분석하고 시각화하는 **Streamlit** 기반의 웹 서비스입니다.

## 🚀 주요 기능

### 1. 성별에 따른 월급 차이 분석
- 남성과 여성의 평균 월급을 비교하여 막대 그래프로 시각화합니다.
- 성별에 따른 경제적 격차를 한눈에 확인할 수 있습니다.

### 2. 나이와 월급의 관계 분석
- 연령대별 평균 월급의 변화를 선 그래프로 보여줍니다.
- 생애 주기에 따른 소득 변화 패턴을 분석합니다.

### 3. 연령대에 따른 월급 차이
- '청년(Young)', '중년(Middle)', '노년(Old)' 층으로 구분하여 소득 수준을 분석합니다.

### 4. 연령대 및 성별 복합 분석
- 성별 임금 격차가 특정 연령대에서 더 두드러지는지 심층 분석합니다.

### 5. 직업별 월급 분석
- 어떤 직업군이 가장 높은 평균 월급을 받는지 상위 10개 직종을 시각화합니다.

### 6. 기타 분석 예정 항목
- 성별 직업 빈도, 종교 유무에 따른 이혼율, 지역별 연령대 비율 등.

## 🛠️ 기술 스택
- **언어**: Python 3.x
- **프레임워크**: Streamlit
- **데이터 분석**: Pandas, NumPy
- **시각화**: Matplotlib, Seaborn
- **기타**: OpenPyXL (Excel 로드용), Pillow (이미지 처리용)

## 📦 설치 및 실행 방법

### 1. 리포지토리 클론
```bash
git clone https://github.com/3679gug/welfare.git
cd welfare
```

### 2. 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 3. 데이터 준비
- `data/welfare_2015.csv` 및 `data/welfare_2015_codebook.xlsx` 파일이 `data/` 폴더 내에 위치해야 합니다.
- (참고: 데이터 파일은 용량 문제로 Git에 포함되지 않을 수 있으니 별도로 준비해 주세요.)

### 4. 서버 실행
```bash
streamlit run python1201.py
```
